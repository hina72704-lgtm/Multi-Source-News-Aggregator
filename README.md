# 📰 Multi-Source News Aggregator

A Python-based application that collects and displays news from multiple sources (NewsAPI and RSS feeds) in one unified interface.

## ✨ Features

- **Multi-Source Integration**: Fetches news from NewsAPI and multiple RSS feeds
- **Category Filtering**: Technology, Sports, Business, Entertainment
- **Search Functionality**: Keyword-based news search
- **Save Articles**: Bookmark interesting articles for later reading
- **Export to CSV**: Export news articles to CSV files
- **Duplicate Prevention**: Smart duplicate detection across sources
- **User-Friendly CLI**: Clean, paginated terminal interface

## 📋 Requirements

- Python 3.7 or higher
- Internet connection
- NewsAPI key (optional, free tier available)
